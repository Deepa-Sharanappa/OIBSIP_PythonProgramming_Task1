# OIBSIP_PythonProgramming_Task1
This project is a Python-based AI Voice Assistant that can perform multiple tasks using voice commands. The assistant listens to the user's voice, processes the command, and responds with speech.
It integrates several Python libraries and APIs to provide features such as web search, weather information, email sending, note-taking, calculations, and more.
The goal of this project is to demonstrate the use of speech recognition, automation, and API integration to build an intelligent assistant.

The objective of this project is to create an AI-powered assistant capable of:
1.Understanding voice commands
2.Performing useful daily tasks
3.Providing information from the internet
4.Automating simple system operations


->Technologies Used:
1.Python
2.SpeechRecognition
3.pyttsx3 (Text-to-Speech)
4.Wikipedia API
5.OpenWeatherMap API
6.SMTP (Email Automation)
7.Requests Library
8.Webbrowser Module
9.Logging Module

->Required Libraries:
Install the following libraries before running the project:
pip install SpeechRecognition
pip install pyttsx3
pip install wikipedia
pip install requests
pip install pyaudio

  Converts speech to text
